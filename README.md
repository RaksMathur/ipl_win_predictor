# IPL Win Predictor 🏏

A machine learning project that predicts the winning probability of an IPL team during a live match.

## Features
- Predicts win probability
- Streamlit-based UI
- Uses trained ML model

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
